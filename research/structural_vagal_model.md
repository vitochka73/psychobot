# Структурно-вагусная модель

## Интеграция полевагальной теории и уровней организации психики

---

## Две оси координат

**Ось X — Вагусное доминирование:**
```
V (вентральный) ←――――――→ S (симпатический) ←――――――→ D (дорсальный)
   безопасность              угроза                  коллапс
   связь                     борьба/бегство          замирание
```

**Ось Y — Уровень организации психики:**
```
НЕВРОТИЧЕСКИЙ (верх)
      ↑
      │
ПОГРАНИЧНЫЙ (середина)
      │
      ↓
ПСИХОТИЧЕСКИЙ (низ)
```

---

## Три уровня организации психики

### НЕВРОТИЧЕСКИЙ УРОВЕНЬ

**Характеристики:**
- Сохранное тестирование реальности
- Стабильная идентичность
- Зрелые защиты (вытеснение, рационализация, сублимация)
- Способность к устойчивым отношениям
- Целостное восприятие себя и других

**Доступ к вагусным состояниям:**
- **V — полный доступ (70-80%)**: Может использовать социальную связь для регуляции
- **S — адекватный доступ (15-25%)**: Мобилизуется при реальной угрозе, быстро восстанавливается
- **D — минимальный (5%)**: Только при экстремальной перегрузке или специфическом триггере

**Гибкость:** Высокая — свободно переключается между состояниями по ситуации

---

### ПОГРАНИЧНЫЙ УРОВЕНЬ

**Характеристики:**
- Тестирование реальности сохранно, но хрупкое (под стрессом может нарушаться)
- Диффузная идентичность ("не знаю, кто я")
- Примитивные защиты (расщепление, проективная идентификация, отрицание)
- Нестабильные интенсивные отношения
- Чёрно-белое восприятие (идеализация/обесценивание)

**Доступ к вагусным состояниям:**
- **V — нестабильный доступ (30-40%)**: Может достигать, но не удерживает. Часто псевдо-V
- **S — доминирует (40-50%)**: Хроническая активация, гипервигильность
- **D — частый (20-30%)**: Легко проваливается при перегрузке

**Гибкость:** Низкая — резкие переключения, "всё или ничего"

---

### ПСИХОТИЧЕСКИЙ УРОВЕНЬ

**Характеристики:**
- Нарушенное тестирование реальности
- Грубое нарушение идентичности (слияние Я/не-Я)
- Самые примитивные защиты (проекция, расщепление на фрагменты)
- Неспособность к стабильным отношениям
- Фрагментированное восприятие

**Доступ к вагусным состояниям:**
- **V — крайне ограничен (5-15%)**: Почти недоступен или доступен только в очень безопасных условиях
- **S — хаотичный (30-40%)**: Дезорганизованная активация, не связанная с реальной угрозой
- **D — доминирует (40-60%)**: Глубокий shutdown, кататония, ступор

**Гибкость:** Отсутствует или хаотична — застревание или непредсказуемые скачки

## Распределение контуров по уровням

### НЕВРОТИЧЕСКИЙ УРОВЕНЬ

| Контур | Типичная картина | Диагнозы |
|--------|------------------|----------|
| **V-V** | Здоровое функционирование с возможными триггерами | Норма, адаптационные реакции |
| **V-S** | Выученная тревога при сохранной базе | ГТР, специфические фобии, лёгкое ОКР |
| **S-S → V** | Тревога со скрытым ресурсом | Тревожные расстройства с хорошим прогнозом |

**Характерно:** Доступ к V-состоянию, способность к терапевтическому альянсу, инсайт

---

### ПОГРАНИЧНЫЙ УРОВЕНЬ

| Контур | Типичная картина | Диагнозы |
|--------|------------------|----------|
| **S-V(p)** | Маска адаптации, скрытая тревога | Высокофункциональное ПРЛ, нарциссизм |
| **S-V(p) → D** | Псевдо-адаптация с коллапсами | ПРЛ, ПТСР, суицидальность |
| **S-S → D** | Тревога → истощение → коллапс | Смешанное, травма |
| **S-D** | Замороженная тревога | Комплексный ПТСР |
| **D-S** | Ажитированная депрессия | Биполярное, ПРЛ с депрессией |
| **V-D → D** | Изоляция, уход | Шизоидное, избегающее |

**Характерно:** Нестабильность, расщепление, псевдо-состояния, резкие переключения

---

### ПСИХОТИЧЕСКИЙ УРОВЕНЬ

| Контур | Типичная картина | Диагнозы |
|--------|------------------|----------|
| **D-D → D** | Глубокий shutdown, ступор | Кататония, тяжёлая депрессия |
| **D-V(p) → D** | Автоматизм → коллапс | Тяжёлая диссоциация, DID |
| **D-S → S** | Хаотичная активация | Острый психоз, мания |
| **S → D (хаотично)** | Дезорганизованные переключения | Шизофрения, острые состояния |

**Характерно:** Потеря связи с реальностью, застревание в D, хаотичные прорывы S

## Клинические импликации

### Диагностика

**Если видим V-V или V-S контур:**
- Вероятно невротический уровень
- Хороший прогноз
- Стандартная терапия эффективна

**Если видим S-V(p) с нестабильностью:**
- Вероятно пограничный уровень
- Нужна работа со структурой
- Риск декомпенсации

**Если видим D-D или хаотичные переключения:**
- Возможен психотический уровень
- Нужна стабилизация
- Медикаментозная поддержка

### Терапевтическая стратегия

| Уровень | Фокус терапии | Работа с вагусом |
|---------|---------------|------------------|
| Невротический | Инсайт, конфликт, смысл | Укрепление V, работа с триггерами |
| Пограничный | Структура, стабилизация, границы | Стабилизация S, развитие доступа к V |
| Психотический | Контейнирование, безопасность, реальность | Выход из D, мягкая активация |

## Движение по уровням

### Декомпенсация (вниз)

```
НЕВРОТИЧЕСКИЙ (V-S) 
      ↓ стресс, травма
ПОГРАНИЧНЫЙ (S-V(p) → D)
      ↓ декомпенсация  
ПСИХОТИЧЕСКИЙ (D-D, хаос)
```

**Маркеры:** Потеря доступа к V, нарастание D, появление псевдо-состояний

### Интеграция (вверх)

```
ПСИХОТИЧЕСКИЙ (D-D)
      ↑ стабилизация
ПОГРАНИЧНЫЙ (S-V(p))
      ↑ терапия
НЕВРОТИЧЕСКИЙ (V-S → V-V)
```

**Маркеры:** Расширение доступа к V, стабилизация переключений, уменьшение псевдо

## Формула структурно-вагусного профиля

Полная формула:

```
[Уровень] : [Контур] → [Реакции]

Примеры:
- N : V-S → (V, S)     — Невротик с тревогой, хороший прогноз
- B : S-V(p) → (S, D)  — Пограничный с псевдо-адаптацией, риск коллапса
- P : D-D → (D)        — Психотический shutdown, нужна госпитализация
```

---

## Ограничения модели

1. **Упрощение:** Реальные люди не укладываются чётко в категории
2. **Динамика:** Уровень может меняться (регресс/прогресс)
3. **Коморбидность:** Один человек может иметь черты разных уровней
4. **Культура:** Нормы "здорового" V различаются в разных культурах
