## 2. Ключевые гипотезы и критерии подтверждения

### H1: Диагностические маркеры

| Гипотеза | Критерий подтверждения | Статистика |
|----------|----------------------|------------|
| S-S-S предсказывает тревогу | AUC ≥ 0.70 | ROC-анализ |
| D-D-X предсказывает депрессию | AUC ≥ 0.70 | ROC-анализ |
| S-V(p)-D предсказывает ПТСР | OR ≥ 3.0 | Логистическая регрессия |
| Псевдо-профили связаны с диссоциацией | r ≥ 0.30 | Корреляция с DES-II |

### H2: Физиологические различия

| Параметр | Контроль vs Тревога | Контроль vs Депрессия |
|----------|--------------------|-----------------------|
| RMSSD baseline | Тревога ↓ | Депрессия ↓↓ |
| LF/HF baseline | Тревога ↑↑ | Депрессия ≈ или ↑ |
| Total Power | Тревога ≈ | Депрессия ↓↓ |
| Реактивность (Δ stress) | Тревога ↑↑ | Депрессия ↓ |
| Восстановление (%) | Тревога 50-70% | Депрессия < 50% |

### H3: Поведенческие корреляты

| Поведенческий признак | Ожидаемая связь с профилем |
|-----------------------|---------------------------|
| Высокий зрительный контакт | V-презентация |
| Монотонный голос | D-презентация |
| Напряжённое тело | S-презентация |
| Несоответствие слов и тела | Псевдо-профиль |

## 3. Что подтвердит / опровергнет модель

### ✅ Модель подтверждена, если:

1. **Распределение профилей** статистически значимо различается между группами (χ², p < 0.01)

2. **S-доминанта** преобладает в группе тревоги:
   - ≥ 50% участников с тревогой имеют S-X-X профиль
   - vs ≤ 20% в контрольной группе

3. **D-доминанта** преобладает в группе депрессии:
   - ≥ 40% участников с депрессией имеют D-X-X профиль
   - vs ≤ 10% в контрольной группе

4. **Псевдо-профили** коррелируют с диссоциацией:
   - r ≥ 0.30 между псевдо-статусом и DES-II

5. **Ретестовая надёжность**:
   - ≥ 70% участников сохраняют тот же профиль через неделю
   - Cohen's κ ≥ 0.60

### ❌ Модель требует пересмотра, если:

1. Распределение профилей НЕ различается между группами (p > 0.10)

2. Физиологические показатели НЕ коррелируют с поведенческой оценкой (r < 0.20)

3. Пороговые значения дают много "пограничных" случаев (> 30%)

4. Ретестовая надёжность низкая (κ < 0.40)

## 4. Возможные сценарии и интерпретации

### Сценарий А: Полное подтверждение
- Все гипотезы подтверждены
- **Вывод:** Модель готова к клиническому применению
- **Следующий шаг:** Лонгитюдное исследование

### Сценарий Б: Частичное подтверждение
- S и V профили работают, D-профили — нет
- **Вывод:** Дорсальный компонент требует пересмотра (косвенные маркеры недостаточны)
- **Следующий шаг:** Добавить поведенческие/субъективные критерии для D

### Сценарий В: Профили работают, пороги нет
- Тренды верные, но чувствительность/специфичность низкая
- **Вывод:** Нужны популяционные нормы и индивидуализация
- **Следующий шаг:** Большая выборка для калибровки порогов

### Сценарий Г: Физиология ≠ поведение
- HRV различается между группами, но не связано с презентацией
- **Вывод:** Компонент Y (презентация) не добавляет ценности
- **Следующий шаг:** Упростить модель до X-Z

### Сценарий Д: Отрицательный результат
- Профили не различаются между группами
- **Вывод:** Модель в текущем виде не валидна
- **Следующий шаг:** Качественное исследование, ревизия теории

## 5. Минимально значимый результат

Для публикации достаточно подтверждения **2 из 5** основных гипотез + хорошая надёжность (κ ≥ 0.60).

Это покажет, что:
1. Профилирование имеет смысл
2. Есть основания для дальнейших исследований
3. Модель может быть уточнена
